# Spotter – Web-Based Fitness Training App

## Description

**Spotter** is an application designed to help users manage and track personalized workout programs. It enables users to create, join, and follow structured training routines while connecting with a wider fitness community. The application emphasizes workout logging, data-driven analytics, and social engagement.

With features such as custom program creation, real-time progress tracking, public workout sharing, and personalized feedback, Spotter aims to support everyone from beginners to personal trainers. The combination of functional parts—Training, Community Exploration, Workout History, and Analytics—ensures a full-circle experience for accountability, discovery, and progress.

---

## Pages Overview

### **Training Page**
Users can:
- Create or join a training program.
- Follow a structured weekly workout plan.
- Log daily workouts with details like reps, sets, weight, and time.
- Track upcoming and completed workouts in a clear, organized layout.

### **Community Page**
Users can:
- Explore and join public workout programs created by other users.
- Follow users and view their public profiles, program contributions, and reviews.
- Share their own programs to benefit the broader community.

### **History Page**
Users can:
- View a calendar-based history of past workouts.
- Revisit specific logs and workout sessions.
- Edit or delete logged workouts.
- Track long-term consistency and habits.

### **Analytics Page**
Users can:
- View lifetime training statistics and weekly summaries.
- Track exercise trends over time.
- Analyze engagement with different muscle groups.
- Identify strengths, plateaus, and growth areas visually.

---

## Core Features

- Create or join custom workout programs.
- Track training sessions on a weekly and daily basis.
- Continue workout plans and log detailed performance data (sets, reps, weights).
- Discover and explore trending programs in the community.
- Follow other users and view shared progress.
- Access visual analytics on workout consistency, muscle targeting, and performance metrics.

## Functional Requirements

### **User Registration and Authentication**
- Secure sign-up and login system.
- User credentials stored securely with encrypted passwords.
- Session handling for persistent login.

### **Workout Program Management**
- Create custom programs with structured weekly and daily breakdowns.
- Define workout types (strength, hypertrophy, endurance, etc.).
- Assign exercises to each day with sets, reps, target muscles, rest times.

### **Workout Logging**
- Log sets with precise inputs (e.g., reps, weight, duration, rest).
- Automatic timestamping of logs.
- Ability to edit or delete logs post-completion.

### **Workout History**
- Calendar-based view of workout logs.
- Quick-access cards showing daily summaries.
- Filters by date, program, or muscle group.
- Edit/view logs with full session breakdowns.

### **Analytics Dashboard**
- Lifetime stats: total workouts, total volume, active days.
- Weekly breakdowns: frequency, exercise performance, intensity.
- Muscle group engagement tracking over time.
- Graphs and charts for easy visualization.

### **Community Interaction**
- Follow and unfollow users.
- Explore trending and most-followed programs.
- Leave and view reviews on shared programs.
- View public user profiles with progress snapshots and shared plans.

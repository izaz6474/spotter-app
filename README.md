# Spotter – Web-Based Fitness Training Application

**Spotter** is an application designed to help users manage and track personalized workout programs. It enables users to create, join, and follow structured training routines while connecting with a wider fitness community. The application emphasizes workout logging, data-driven analytics, and social engagement.

With features such as custom program creation, real-time progress tracking, public workout sharing, and personalized feedback, Spotter aims to support everyone from beginners to personal trainers. The combination of functional parts—**Training**, **Community Exploration**, **Workout History**, and **Analytics**—ensures a full-circle experience.

---

## 🌟 Pages Overview

### 🏋️‍♂️ Training Page
Users can:
- Create or join a training program
- Follow structured weekly workout plans
- Log exercises with reps, sets, weights, and time
- View upcoming and completed workouts

### 🌐 Community Page
Users can:
- Discover public workout programs created by others
- Join programs shared by the community
- Follow other users and view their profiles
- Read and write reviews for programs
- Share their own programs to help others

### 📅 History Page
Users can:
- View a calendar of all past workouts
- Revisit, edit, or delete old workout logs
- Track consistency and identify gaps over time

### 📊 Analytics Page
Users can:
- Access lifetime training statistics
- View weekly workout summaries
- Track progress over time through comparisons
- Visualize muscle group engagement
- Analyze trends in specific exercises

---

## 🔧 Core Features
- Create and manage custom workout programs
- Track training sessions in a weekly/day-wise layout
- Continue a saved program and log real-time data (sets, reps, weights)
- Visual analytics including total weight lifted, streaks, and muscle focus
- Explore trending programs and creators in the community
- Social engagement through following, sharing, and reviews

---

## 🔁 User Flow
1. Users land on the **Train** page.
2. They can onboard into a program or create a new one.
3. Programs are structured into weeks, days, and exercises (with target muscles).
4. Completed workouts are logged and reflected in both **History** and **Analytics**.
5. Users can explore community programs and follow other members.

## ✅ Functional Requirements

### 🧾 User Management
- User registration and authentication system
- Secure login with session persistence
- Ability to update profile and manage credentials

### 📦 Workout Program Management
- Create custom workout programs with:
  - Program title, description, duration, difficulty level
  - Weekly and daily structuring of workouts
  - Exercises tagged with sets, reps, weights, rest time, and target muscle groups
- Option to publish programs publicly or keep private

### 📅 Workout Logging
- Real-time logging of workouts
- Capture performance data: reps, sets, weight, time, notes
- Ability to mark workouts as completed or skipped
- Auto-sync logs to calendar and analytics

### 🔁 Program Onboarding
- Join programs directly from the Community
- Seamlessly switch between current and joined programs
- Resume workouts from where the user left off

### 🗓️ History Tracking
- Calendar view with marked workout days
- Detailed log view for any selected day
- Edit or delete logs

### 📈 Analytics Dashboard
- Lifetime stats: total volume lifted, sessions completed, etc.
- Weekly summaries: progress snapshot, consistency graph
- Muscle group tracking
- Exercise performance trends and progression charts

### 🌐 Community Interaction
- Explore public programs based on popularity, reviews, or category
- Follow users and see their public activity
- Review and rate programs
- Save and clone community programs for personal use
